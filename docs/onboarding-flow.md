# Onboarding Flow (v0.1)

1. Sign up (email / magic link)
2. Connect Slack (recommended) OR Telegram (daily brief only)
3. Connect Gmail (read-only)
4. Done in 30 minutes → start working inside messenger

Toggles (v0.1 only):
- Memory: ON
- Meeting summary → action items: ON
- External send: OFF (always)
- Daily brief: ON
